Start with these bundled elements, then use one or as many as you want in any combination. When you want more than the starter set, the Collection is the next step.

What this gives you on day one

A real starting point

You can activate something useful immediately instead of facing an empty portfolio and a blank prompt.

Examples across every type

The starter set teaches the model by example, showing how each element type is structured and how they combine.

Dollhouse expert suite included

The bundled expert ensemble pairs the expert persona with knowledge-base memory for guided help with the platform itself.

Session monitor included

The starter agents include a session monitor that helps keep the LLM synchronized with external state changes.

The starter portfolio lives locally in ~/.dollhouse/portfolio/. From there you can activate, edit, sync to GitHub, or submit new work back to the Collection. You can also browse the Collection directly for the broader public catalog.
